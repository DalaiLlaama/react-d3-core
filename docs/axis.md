# Axis

Building Axis in this component.  Here is where xaxis and yaxis and grid system being built.

## Import

```js
import {
  Axis as Axis
} from 'react-d3-core';
```

## Setting props

You can customize `Axis` component using the following properties.

### Basics

- type
- showAxis

### Scale

- scale
- range
- domain
- rangeRoundBands

### Tick

- tickOrient
- tickFormat
- tickPadding
- innerTickSize
- outerTickSize
- ticks
